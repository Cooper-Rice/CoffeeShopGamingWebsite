# CoffeeShopGaming Website

Unofficial website code for the CoffeeShopGaming website.

A redesign of [voidblock6's original](https://github.com/voidblock6/CoffeeShopGaming-Website), built with plain HTML, CSS, and JavaScript. No frameworks, no build steps.

## Pages

- `index.html` — Home page with server info and team
- `library.html` — Picture gallery
- `contact.html` — Contact form (opens Gmail)

## Files

```
index.html
library.html
contact.html
styles.css
theme.js
images/
```

## Features

- Dark/light mode toggle (persists across pages)
- Hamburger menu on mobile
- Contact form that opens Gmail compose
- No dependencies
